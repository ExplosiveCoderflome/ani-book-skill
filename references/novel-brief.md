# 新书简报合同

`novel-brief.md` 是一本小说的书级创作合同。它把模糊灵感转成后续世界观、角色、卷规划和章节写作都可消费的方向；它不是要求新手先填写完的长表单。

先确认“最小开书集”，再由 AI 给出推荐值并逐步深化其余设定。每一项标注为 `用户确认`、`AI 推荐` 或 `待探索`。用户确认的方向受保护；AI 推荐可以在明确说明影响范围后调整。

## 最小开书集

- 一句话灵感或 premise；
- 读者频道倾向：男频、女频、双向/泛读者，或“不确定，由 AI 推荐”；
- 题材与时代/现实距离；
- 主角的核心欲望和最直接阻碍；
- 希望读者持续获得的核心回报；
- 预计篇幅：短篇/中长篇/长篇，或大致章节数。

没有足够信息时，不阻塞开书：给出一套推荐值，说明它如何影响钩子、角色、节奏和章节预算，并让用户优先只确认最有感觉的一两项。

## 完整模板

```markdown
# <书名>｜新书简报

## 定位与读者

- 书名 / 暂定名：
- 创作模式：原创 / 续写 / 改编（来源与授权边界）
- 发布形态与平台倾向：
- 读者频道倾向：男频 / 女频 / 双向或泛读者 / 由 AI 推荐
- 目标读者与阅读场景：
- 题材、子题材与时代/现实距离：
- 一句话 premise：
- 阅读承诺：读者为什么要读这本书，并愿意持续追读什么：

## 故事发动机

- 主角幻想、核心欲望与主动手段：
- 稀缺杠杆（能力、资源、关系或身份）：
- 核心冲突与主要阻碍：
- 可重复升级的行动—回报—代价循环：
- 长线谜团/关系线/成长线：
- 第一章压力、可见变化与章末牵引：

## 叙事与读感

- 叙事视角：第一人称 / 第三人称有限 / 第三人称全知 / 多视角（切换边界）：
- 语气与语言质感：
- 节奏偏好：快 / 均衡 / 慢热；每 3—5 章的可见回报类型：
- 情绪浓度：克制 / 中等 / 强烈；主要情绪体验：
- 关系与感情线的比重及边界：
- 爽点、悬念、成长、破案、经营、冒险等回报优先级：
- 希望避免的套路、叙事方式或内容边界：

## 篇幅与生产节奏

- 预计总章节数或范围：
- 预计总字数（可由章节数 × 单章长度推算）：
- 正文字数统计口径：默认不含标题、Markdown 标记与审查记录；
- 书级单章长度档：短章 / 标准章 / 长章，或自定义；
- 单章目标字数与可接受区间：
- 默认篇幅档建议：短章 `1500–2200`、标准章 `2500–3500`、长章 `3800–5000`；题材、平台和用户偏好优先于建议；
- 哪些章节允许覆盖默认值：开篇、高潮、审讯、战斗、情感收束或多线交叉等，并写明允许范围：
- 单章最低有效内容：不得为达到字数而牺牲的事件、回报、转折与钩子：
- 超短/超长的处理偏好：补写有效场景 / 删减重复 / 保留并记录原因：
- 前期/当前卷计划深度：
- 预期更新节奏或每次生成批次：
- 允许调整的空间：章节数 / 单章长度 / 卷数 / 节奏，及其优先级：

## 世界与角色起点

- 世界形态与必须遵守的初始限制：
- 主角起点身份、社会位置与可见弱点：
- 核心关系承诺或冲突来源：
- 必须保留的真实素材、作者选择或既有正文：

## 设定来源与确认记录

| 设置 | 当前值 | 来源（用户确认 / AI 推荐 / 待探索） | 影响范围 | 可在何时再确认 |
| --- | --- | --- | --- | --- |
| 读者频道倾向 |  |  | 阅读承诺、角色关系、回报组织 | 卷战略前 |
| 叙事视角 |  |  | 正文视角与信息边界 | 第一章前 |
| 预计章节数 |  |  | 卷战略与 hard/soft 规划 | 卷战略前 |
| 单章长度 |  |  | 章节合同与生成预算 | 首章前 |
```

## 设定如何进入后续生产

- 读者频道倾向不是刻板的性别规则；它只帮助 AI 组织读者回报、关系比重、情绪和卖点，并始终服从用户明确的题材与阅读承诺。
- 预计总章节数是规划区间，不是强制平均分配。卷战略仍按阶段承诺、局面升级和卷末牵引判断；用户固定章节数时必须保护该决定。
- 单章目标字数必须带可接受区间、统计口径和偏差处理规则。篇幅不足时推进有效事件、冲突、对话或行动，不能用回顾和空泛心理描写填充；超过上限时优先删去重复解释和无后果段落，而不是砍掉已成立的转折。
- 叙事视角、内容边界和必须避免的套路进入章节计划与上下文包，作为写作和审查共同遵守的约束。
- 更新节奏只影响生产批次和规划粒度，不得为了赶更牺牲章节合同、正文可读性或连续性回灌。

## 验收

- 最小开书集能清楚说明这本书为谁而写、主角为何行动、读者会获得什么。
- 总章节预算与单章目标长度均可执行，且允许在卷战略前被明确调整。
- 叙事视角、节奏、情绪和内容边界能够指导第一章，而不是停留在抽象形容词。
- 所有 AI 推荐值都标明为推荐，未伪装成用户已确认的决定。
